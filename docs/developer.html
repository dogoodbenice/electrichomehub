GET /api/devices

Retrieve all registered devices with warranty and status information

{
  "devices": [
    {
      "id": "device_123",
      "name": "Smart Refrigerator",
      "manufacturer": "Samsung",
      "model": "RF28R7351SR",
      "warrantyStatus": "active",
      "location": "Kitchen"
    }
  ]
}
POST /api/devices

Register a new device in the Electric Home Hub platform

{
  "name": "Smart Thermostat",
  "manufacturer": "Nest",
  "model": "T3007ES",
  "serialNumber": "ABC123456",
  "location": "Living Room",

GET /api/alerts

Fetch active monitoring alerts and notifications

{
  "alerts": [
    {
      "id": "alert_456",
      "type": "warranty_expiring",
      "severity": "medium",
      "deviceId": "device_123",
      "message": "Warranty expires in 30 days"
    }
  ]
}
POST /api/documents

Upload device documentation and manuals

{
  "deviceId": "device_123",
  "type": "manual",
  "category": "user_guide",
  "file": "base64_encoded_content",
  "filename": "manual.pdf"
}
